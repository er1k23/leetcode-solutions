# LeetCode / NeetCode Practice

This repository contains my solutions to algorithm and data structure problems, solved while following the NeetCode roadmap (https://neetcode.io/roadmap) as part of my preparation for technical interviews.

Each solution is written independently: I read the problem statement first, think through the approach without looking at hints or discussions, implement the solution, test it locally with my own test cases, and only then submit it. After a solution is accepted, I review alternative approaches and rewrite my solution from scratch to make sure I actually understand it.

## Structure

Solutions are organized by topic:

Topic Name/
    NNN Problem Name/
        solution.py
        README.md

Each problem folder contains the solution file and a short note on the approach, time and space complexity.
